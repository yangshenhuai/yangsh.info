Julien Danjou

What's OpenStack?

  • An open source cloud computing platform (Apache 2 licensed)
  • A project started in 2010 via Rackspace and NASA (sort of)
  • First goal: provides IaaS (Infrastructure as a Service)
  • 200+ companies in the project
  • 9 integrated projects (components), and growing to be more than Iaas

Architecture

Architecture: SQL powered

PostgreSQL, is it working with OpenStack?

  • In theory, yes
  • In practice, it has often been broken and sometimes for week (non-voting gate check)
  • Primary target is MySQL in cluster mode (i.e. Galera)
  • Large community of ex-MySQL folks

Technologies?

Python + SQLAlchemy =

  • We target the lowest denominator: SQLite: you can't use any advanced features of any other RDBMS
  • We rely on an ORM to handle the queries, etc
  • ORM allows you to express things that works on certain RDBMS, and bugs on other
  • Many Python developers do not have a clue on how a RDMBS work

Conclusion

  • RDBMS is a central point of many OpenStack components
  • Wrongly used in a lot of cases
  • The community in general lacks a proper understand of how it should be used and leveraged
