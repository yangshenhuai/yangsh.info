--- title: "java8 in a Nutshell--Stream API" created: !!timestamp '2016-1-21 23:20:00' image: /media/images/blog/2016/streams.png tags: - Java --- {% block excerpt %} {% mark excerpt %} Stream are the key abstraction in Java8 for processing collections of value. Streams follow the "what,not how" priciple,you don't need to specify how to get the data. you just specify how the computation should work. {% endmark %} {% endblock %} # Stream Creation for collection,you can call stream(or parallStream) method to create a stream {% syntax Java %} List words =... long count = words.stream().filter((s) -> s.length()>2).count(); {% endsyntax %} or create with Stream.of method {% syntax Java %} Stream stream = Stream.of("A,B,C".split(",")); Stream stream = Stream.of("A","B","C"); Stream empty = Stream.empty(); // empty Stream {% endsyntax %} and you can create a infinite stream {% syntax Java %} Stream.generate(generate(Supplier s)); //for example Stream randoms = Stream.generate(Math::random) {% endsyntax %} {# Local Variables: #} {# mode: markdown #} {# End: #}
