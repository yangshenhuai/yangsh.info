--- title: "A Byte Of Go(1)" created: !!timestamp '2017-01-08 23:21:00' image: /media/images/blog/2017/go_lang_1.jpg tags: - Go --- {% block excerpt %} {% mark excerpt %} [go](https://golang.org/) is a open source programming language created by Google.it is strongly typed ,compiled and has explicit support for concurrent programming {% endmark %} {% endblock %} #Install Go lang is open-source , so you can dowload from its [offical website](https://golang.org/dl/) , the latest version is 1.7.4 you can also start with go with the [sandbox](https://tour.golang.org/welcome/1) #Hello world Here is a hello world program with go. this program is in `main` package , by convention , the package name is same as the last element of the import path. Here import `fmt` package. package `fmt` implements formatted I/O with functions analogous to C's printf and scanf. {% syntax shell %} package main import "fmt" func main(){ fmt.Println("Hello world") } {% endsyntax %} #Imports and Exports In `Hello world`, you can see how to import a package, you can import multiple packages like below {% syntax shell %} import( "fmt", "math" ) {% endsyntax %} In Go , a name is exported automaticlly if it begins with a capital letter, for example {% syntax shell %} package main import ( "fmt", "math" ) func main(){ fmt.Println(math.Pi) // it cannot run succcessfully if you write math.pi,because math.pi is not a exported name } {% endsyntax %} #Functions A function can take zero or more arguments ,and have zero or more returns, for example `add` function below (the type is after the argument and function name) {% syntax shell %} package main import "fmt" func add(x int, y int) int{ return x+y } func main(){ fmt.Println(add(3,4)) } {% endsyntax %} as both x , y is int type , you can write like this `func(x,y int)` and the you can also have multiple returns in single function {% syntax shell %} package main import "fmt" func addAndMinus(x,y int)(int,int){ return x+y,x-y } func main(){ s,m := addAndMinus(3,4) fmt.Println(s,m) } {% endsyntax %} if you have multiple returns , you may name the return values to avoid the confusion {% syntax shell %} package main import "fmt" func addAndMinus(x,y int)(sum int, minus int){ sum = x+y minus = x-y return } func main(){ s,m := addAndMinus(3,4) fmt.Println(s,m) } {% endsyntax %} #Variables(TODO)
