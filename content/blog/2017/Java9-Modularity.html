--- title: "Java9 模块化" created: !!timestamp '2017-08-31 12:21:00' image: /media/images/blog/2017/module.jpg tags: - Java --- {% block excerpt %} {% mark excerpt %} Java9 今年九月份就要发布了，这次发布最大的change就是模块化，这也可能是Java问世以来改变最大的一次版本发布.因为整个jdk都被模块化了 之前 "大而全" 的rt.jar(runtime)终于寿终正寝,取而代之的是很多个相对独立模块.模块化的Java提供了`更强的封装`,使程序员使用`定义良好的接口`来相互调用 并且要`显示声明依赖`,这些都能保证开发中避免遇到不必要的错误. {% endmark %} {% endblock %}
